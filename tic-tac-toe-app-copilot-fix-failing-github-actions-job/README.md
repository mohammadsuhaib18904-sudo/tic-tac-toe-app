# Neon Tic Tac Toe

A neon-themed Tic Tac Toe game with smooth animations and haptic feedback.

## Features
- Neon-themed UI with glowing X and O symbols
- Smooth animations and visual effects
- Haptic feedback for moves and game events
- Player current turn indicator
- Win detection with highlighted winning line
- Draw detection
- Game reset functionality
- Left-handed mode setting

## How to Run
1. Ensure you have Flutter installed (https://flutter.dev/docs/get-started/install)
2. Clone or download this project
3. Run `flutter pub get` in the project directory
4. Connect a device or start an emulator
5. Run `flutter run` to start the app

## Customization
You can customize:
- Colors by modifying the color values in the widgets
- Animation timing by adjusting durations
- Game settings through the settings provider
- Haptic feedback preferences
